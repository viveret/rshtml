Project Summary

This project is focused on imitating Razor / Blazor and the MVC system for C # / .NET. There are a lot of similarly or equally named classes, enums, and methods, but 99% of the code was written without referencing the dotnet core source code.

Supported Features

  • API-only and full web applications
  • Dependency injection / type constructor dependency resolution
  • Service collections and scoping
  • HTTP Request pipeline system
  • Controllers, actions, action results
  • HTML view templating system

Main Differences from C# / dot net

  • Razor templating text nodes cannot capture spacing at the moment. Non-captured spaces are automatically stripped To fix this, use @"" or @format!().
  • Rust HTML (.rshtml) files are not known by the cargo build system. Changes won't automatically trigger rebuilds.
